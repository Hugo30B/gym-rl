# Bala2 RL

Este repositorio contiene código **modificado** del código oficial de M5Stack para el **Bala2 Fire**.

Se trata de una versión adaptada del firmware oficial de M5Stack para el robot Bala2 Fire, con cambios para aplicar aprendizaje por refuerzo en vez de un modelo matemático.

## Origen

- **Código base**: M5Stack Official Repository
- **Robot**: Bala2 Fire
- **Modificaciones**: Remodelación por completo del sistema de balanceo por una función de inferencia de una red neuronal.

## Notas

El código mantiene la estructura y funcionalidad principal del proyecto oficial, con las adaptaciones específicas que hemos hecho.
